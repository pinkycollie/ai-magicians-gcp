# Contributing to 360 Magicians

We welcome contributions to this project! By participating, you agree to abide by our code of conduct.

## How to Contribute

1. Fork the repository and create a new branch.
2. Make your changes and write tests if applicable.
3. Run tests to ensure everything works correctly.
4. Open a pull request with a clear description of your work.

## Reporting Issues

If you find a bug, please create an issue and include:
- A clear description of the problem.
- Steps to reproduce the issue.
- Any related logs or error messages.

Thank you for making this project better!
