# Quick Start Guide

## For AI Studio Users

### Upload These 3 Files:
1. `index.html` (6.3 KB)
2. `app.js` (19 KB)  
3. `metadata.json` (395 bytes)

### Steps:
1. Go to https://aistudio.google.com/
2. Click "New App" or "Create"
3. Upload the 3 files above
4. Click "Preview" to test
5. Click "Share" to distribute

**That's it!** Your app is ready to use.

## For Local Testing

### Option 1: Python
```bash
python -m http.server 8000
```

### Option 2: Node.js
```bash
npx http-server -p 8000
```

### Then:
Open http://localhost:8000 in your browser

**Note:** For local testing, you need to add your Gemini API key in `app.js`.

## Common Issues

**App doesn't load:**
- Check browser console for errors
- Verify all 3 files are uploaded
- Ensure you have a Gemini API key

**Chat not working:**
- Make sure your API key is valid
- Check your API quotas
- Verify internet connection

**Styling looks wrong:**
- Clear browser cache
- Try a different browser
- Reload the page
